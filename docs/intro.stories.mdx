# 1. Introduction

**@electinth/common** is a collection of ELECT assets including

- [Web Components](https://developer.mozilla.org/en-US/docs/Web/Web_Components) written with [Hybrids](https://hybrids.js.org) using Typescript
- Fonts with coresponded stylesheet
- Guides on the left panel to help you set everything up
